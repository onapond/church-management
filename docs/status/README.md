## 시스템 개요

- **앱 이름**: 청파중앙교회 교육위원회 관리 시스템
- **제품 방향**: 교육부서 행정 관리에서 Church OS로 확장
- **기술 스택**: Next.js 16.1.6, React 19, TypeScript strict, Supabase, TanStack Query, Tailwind CSS v4
- **배포 URL**: https://church-opal.vercel.app
- **GitHub**: https://github.com/onapond/church-management

## 주요 기능 요약

1. **출결 관리** - 부서/셀별 예배 및 모임 출결 체크, 엑셀 내보내기, 통계 연동
2. **보고서 시스템** - 주차/모임/교육/셀 리더 보고서 작성, 결재, 피드백, 사진 표시
3. **교인 관리** - 교인 명단, 다중 부서 소속, 셀 배정, 프로필/일괄 사진 관리
4. **회의 관리** - 회의 등록, 구조화 회의록, PDF 첨부, 수정/등록 취소, 회의 피드백
5. **회의 사전 안건** - 부서별 안건/질문/공지, 댓글, 인라인 수정, PDF 첨부
6. **회계 관리** - 부서별 장부, 수입/지출 입력, 지출결의서, 엑셀 가져오기/내보내기
7. **심방/사진/통계** - 심방 일정, 활동 사진, 출결/부서 통계
8. **알림과 권한** - 인앱/웹푸시 알림, 관리자 승인 기반 가입, 역할/부서/RLS 기반 접근 제어

## 최근 반영된 운영 변경

- 회의록 원본 PDF 첨부와 회의 상세 PDF 보기
- 회의 사전 안건/질문/공지, 댓글, 편집, 부서 안건 PDF 첨부
- 보고서 사진 표시와 명시적 제출 가드
- 일반 셀 리더의 peer 보고서 열람 제한
- CU1 셀 배정/셀명 변경 등 데이터-only 운영 SQL 기록
