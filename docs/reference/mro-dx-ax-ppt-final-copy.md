## 슬라이드 2. 핵심 사항 요약

### 제목
핵심 사항 요약

### 본문
- `Next.js + Supabase + TypeScript` 기반 운영 시스템 구축
- 출결, 보고, 승인, 회의, 회계, 알림 기능 통합
- 역할 기반 권한과 데이터 접근 통제 적용
- AI를 바로 넣기보다, 나중에 붙일 수 있는 구조로 설계
- MRO 환경의 구매, 승인, 공급사 협업 업무로 확장 가능

## 슬라이드 5. AX 기능 1

### 제목
AX 기능 1. 구조화된 회의록 기반 AI 준비

## 슬라이드 9. 기술 구조 요약

### 제목
기술 구조 요약

### 본문
- Frontend: `Next.js 16`, `React`, `TypeScript strict`
- Backend: `Supabase`, `PostgreSQL`, `Auth`, `RLS`
- Data Layer: `TanStack Query`
- Notification: `web-push`
- Deployment: `Vercel`
- 운영 시스템에 맞춘 구조적 설계와 확장성 확보

결론
